# Common Steps
- Install the preffered game.
- Install Creation Kit for your game.
- Open "<Your Game Folder>/Data/"
- Unpack Scripts.zip manually.
- Now cut all files from "<Your Game Folder>/Data/Scripts/Source/Scripts/"
- And paste to "<Your Game Folder>/Data/Scripts/Source/"
- Delete empty Scripts folder "<Your Game Folder>/Data/Scripts/Source/Scripts/".

# Using Bottles
1. Create a Bottle.
2. Add shortcut for ckpe_loader.exe
> [!Note]
> If you launch ckpe_loader right now, you will only run base Creation Kit.
3. Go to Options>Settings>Compatibility>DLL Overrides
4. And add a new override called `winhttp`
> [!Note]
> Now you can launch ckpe_loader, and use CKPE.
